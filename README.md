AzPainter for Mac
====

Convert the AzPainter to run on Mac

## Requirement
* brew
* x window system environment(XQuartz etc)

## Install
It will be installed in the `/Applications`

```zsh
$ brew install jpeg makeicns
$ make
$ make install
```
